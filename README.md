## Ember launcher

Run Ember with:

```bash
npm run ember
```

What this does:

- Starts the Ember agent framework (runtime + web communication UI).
- Agent runtime listens on port `4317` and stores state in `~/.ember-agent`.
- Uses `PORT` (default `3005`) and prints the URL/port on startup.
- Prints the URL, port, and a `[● RUNNING]` indicator.
- Keeps the process running until you stop it.
- On `Ctrl+C`, it cleanly shuts down both runtime and web server.
